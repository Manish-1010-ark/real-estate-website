## üöÄ Quick Start

```bash
# Clone and setup
git clone https://github.com/Manish-1010-ark/real-estate-website.git
cd real-estate-website/website

# Install dependencies
pip install -r requirements.txt

# Setup database
flask db upgrade

# Run the application
flask run
```

Visit `http://localhost:5000` to view the site.
